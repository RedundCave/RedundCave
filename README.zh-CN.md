## What is RedundCave?

“熔洞”（RedundCave）旨在充分利用点对点网络的潜力，通过异构冗余和边缘存储技术，为用户数据提供坚如磐石的安全保障。 在此系统中，每位用户不仅是数据存储节点，更是保护和支持整个网络运作的一份子，数据将在广阔的设备网络中自由流动。
